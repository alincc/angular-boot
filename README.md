Packed Angular 2.x with TypeScript, including dependencies to have a minimal playground.

### Usage
run `npm install fangular --save`

### What it contains
- angular2 (2.0.0-beta.3)
- angular2-polyfill (0.0.2)
- es6-shim (^0.33.3)
- rxjs (5.0.0-beta.0)
- systemjs (0.19.6)
- typescript (^1.7.5)
- zone.js (0.5.11)
